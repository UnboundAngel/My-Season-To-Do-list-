# Theme reference

| Theme | Palette | Highlights |
| --- | --- | --- |
| Professional Dark | Accent `#7c5bff`, Gradient shadows | Neutral backdrop with a soft violet highlight for everyday use. |
| Halloween | Accent `#ff914d`, Pumpkin glow | Pumpkin path along the footer, warm lantern overlay, and candy-corn gradients. |
| Christmas | Accent `#8ef1ff`, Frosted blues | Snow field with evergreen silhouettes, dark midnight sky, and Santa sleigh trail. |
| Mocking Neon | Accent `#14ffe9`, Magenta echo | Neon grid overlay with high-contrast buttons for playful mock sessions. |

Holiday overlays can be toggled off via the header checkbox if you prefer the base palette without the thematic glow.
